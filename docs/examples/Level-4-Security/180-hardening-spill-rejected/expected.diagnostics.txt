FUNGI-HARDEN-005 (RESIDENCY_CEILING_UNHONOURABLE)

An explicit `hardening { residency register_only }` ceiling cannot be honoured by
`host mlock_posix` (mlock guarantees no-swap/no-disk, not register pinning). Fail-closed:
the value is REJECTED rather than silently spilled past its ceiling (RD-0358 H-2 / HV5).
Declare a capable seam (e.g. `host register_pinned`) or relax the ceiling with an audited opt-out.
